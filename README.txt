Description:
on souhaite créer un programme java qui permet de gérer le personnel et les étudiants d'une école.
Pour cela on créera les classes Etudiant, Employé, Professeur, Personne avec les règles suivantes:  
 	La classe Etudiant hérite de la classe Personne.
 	La classe Professeur hérite de la classe Employe et la classe Employe hérite de la classe Personne.
	Un Etudiant est une Personne.
	Un Professeur est un Employe et un Employe est une Personne.
	Une Personne a un nom, un prénom et un id.
	Un Etudiant a un matricule.
	Un Employé a un saliare.
	Un Professeur a une spécialité qu'il enseigne.

Travail à faire 
Développer les classes dans le package "ecole".
Chaque classe doit contenir un constructeur d'initialisation et une méthode.
Développer une classe App dans le package "test", dans cette classe on demande de créer : 
deux étudiants.
deux employés.
deux professeurs.
afficher les informations de chaque personne.
